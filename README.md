# codex-cfw
CloudFlare Worker Discord bot

## Dev Setup

### Setup environment
Copy `example.env` -> `.env` and define variables

### Register commands
```
npm run register
```

### Run server
```
npm start
```

### Run ngrok
```
npm run ngrok
```

### Setup ngrok endpoint
Set `Interactions Endpoint URL` on Discord application settings to `Forwarding` URL in ngrok
